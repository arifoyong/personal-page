## AI-Powered Interactive Resume

An interactive personal résumé powered by Generative AI — designed to represent me authentically and professionally.
This project showcases how AI can transform traditional résumés into dynamic, conversational experiences.


## Overview
This website uses a custom AI agent that responds on my behalf, providing insights into my background, professional experience, and skills. It aims to create a more engaging way for visitors to learn about me.

## Features
- Conversational AI Representation
The AI can answer questions about my career, background, and expertise — acting as a faithful, professional extension of myself.

- Real-time Streaming Responses
Built with streaming output for a smooth, responsive chat experience.

- Smart Notifications
Integrated with Pushover to notify me when:
  1. A question couldn’t be answered by the AI
  2. A visitor expresses interest in connecting with me
